# 出席クイック

千葉工大の出席サイト (`attendance.is.it-chiba.ac.jp`) に、時間割から現在の講義を判定してワンタップで飛べる Web アプリ。

## 機能

- 時間割の登録・編集（曜日・時限・講義名・教室番号）
- 現在の講義を判定して大きなボタンで出席ページへのリンクを表示
- 次の講義・本日の時間割一覧
- 週間時間割ビュー
- 時限の時間設定（1限〜5限のデフォルト入り、変更可）
- データのエクスポート / インポート（JSON）

## 使い方

1. ブラウザで [GitHub Pages URL] を開く
2. 「設定」タブから講義を追加（講義名・曜日・時限・教室番号）
3. スマホのホーム画面に追加（PWA 対応）しておくと起動が速い
4. 講義開始時刻になると「今の講義」カードが表示されるので、ボタンをタップして出席サイトへ

## データ保存

- ブラウザの localStorage に保存（端末内のみ）
- 設定 → エクスポート/インポートで端末間移行可能

## 開発

静的ファイルのみ。ローカル確認は任意の静的サーバで:

```sh
python3 -m http.server 8000
# → http://localhost:8000
```
